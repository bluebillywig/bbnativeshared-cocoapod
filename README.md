# BlueBillywigNativeShared-iOS

Shared business logic framework used by [BlueBillywigNativePlayerKit-iOS](https://github.com/bluebillywig/bbnativeplayerkit-cocoapod). This module contains the cross-platform core — controllers, data models, networking, and event handling — that powers the Blue Billywig Native Player SDK on iOS.

> **This is an internal dependency.** You do not need to install it directly. It is automatically included when you install `BlueBillywigNativePlayerKit-iOS` via CocoaPods or Swift Package Manager. See the [PlayerKit repository](https://github.com/bluebillywig/bbnativeplayerkit-cocoapod) for integration instructions.

## Requirements

- iOS 14.0+

## Installation

### CocoaPods

Installed automatically as a dependency of `BlueBillywigNativePlayerKit-iOS`. If needed separately:

```ruby
pod 'BlueBillywigNativeShared-iOS'
```

### Swift Package Manager

Resolved automatically when adding `BlueBillywigNativePlayerKit-iOS`. If needed separately, add:

```
https://github.com/bluebillywig/bbnativeshared-cocoapod
```
